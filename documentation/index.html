Introduction

Niwax – is a modern and creative HTML5 Multi-purpose template that is created for IT Companies, Marketing Agency, Startup, Freelancer Portfolio, Agency Portfolio, Design Studio, Marketing Landing Page, Web Design, Web Development, Graphic Design, App Development, Digital Marketing, Lead Generation, and any other online business solutions!. It comes with modern design and a light blazing fast loading template.

Theme Help

Add Dark Mode

Add active-dark class in body


Change mega menu class for dark mode- nav-bg-w to nav-bg-b

nav-bg-w for light background

nav-bg-b for dark background


Add Dark Mode in section

Use light-dark and deep-dark class to change section or div background-color to dark.


Change Logo

Use the white color logo in dark mode for better visibility


How to Change Gradient Background

Choose the section you want to change background gradient.

Right click on the section and open inspact tool (ctrl+shift+i)
